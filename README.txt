Python version: 2.7.13
Prerequisites for running this program:
- Python version 2.7.13 installed.
- The news database installed on the virtual box using vagrant.
- git bash.

Purpose of the program: Display the result of the following questions:
1. What are the most popular three articles of all time?
2. Who are the most popular article authors of all time? 
3. On which days did more than 1% of requests lead to errors?

In order to run the program, you will need to run navigate to the location
where your vagrant machine and the news database is installed from git bash home directory.
From there, please enter the following command:
python log_analysis.py. 

Files that should be included in the program
- log_analysis.py
- README.txt
